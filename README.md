# Gestion de parc informatique - it_parc

Ce module personnalisé Odoo 18 est développé pour les besoins internes de TECHPARK CI.
Il permet la gestion centralisée du parc informatique (matériels, affectations, interventions, et contrats).

## Fonctionnalités Principales
- **Centralisation** : Référentiel unique des équipements.
- **Traçabilité** : Suivi des affectations (employés, départements) et de l'historique des maintenances.
- **Anticipation** : Alertes automatiques pour l'expiration des garanties et contrats.
- **Reporting** : Rapports PDF et exports Excel complets.
- **Pilotage** : Dashboard interactif en OWL.

## Installation
1. Copier le dossier `it_parc` dans le répertoire des addons de votre instance Odoo 18.
2. Mettre à jour la liste des applications (`Mettre à jour la liste des applications`).
3. Chercher `it_parc` et cliquer sur Installer.
